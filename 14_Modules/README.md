# 14_Modules

Modules let you reuse code and use built-in Python functionality.

## What you will learn
- Importing modules
- Using math, random, and datetime
- Creating and importing your own module

## Files in this chapter
- math_module.py: using the math module
- random_module.py: generating random values
- datetime_module.py: working with dates and times
- custom_module.py: importing a custom module

## Practice ideas
- Create a helper module for common math tasks.
- Build a small guessing game with random numbers.

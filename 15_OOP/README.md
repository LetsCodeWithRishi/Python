# 15_OOP: Object-Oriented Programming

Object-oriented programming organizes code into classes and objects.

## What you will learn
- Classes and objects
- Constructors
- Inheritance
- Polymorphism
- Encapsulation
- Abstraction

## Files in this chapter
- class_object.py: basic class and object example
- constructor.py: initializing objects with __init__
- inheritance.py: reusing behavior through inheritance
- polymorphism.py: different classes using the same interface
- encapsulation.py: protecting data inside objects
- abstraction.py: defining abstract base classes

## Practice ideas
- Create a Student class.
- Build a BankAccount class.
- Design a LibraryBook class.

# 09_Tuples

Tuples are similar to lists, but they are immutable and are useful when values should not change.

## What you will learn
- Tuple creation
- Accessing tuple items
- Packing and unpacking
- Tuple methods

## Files in this chapter
- tuple_basics.py: basic tuple usage
- packing.py: tuple packing
- unpacking.py: tuple unpacking
- methods.py: using tuple methods

## Practice ideas
- Store coordinates in a tuple.
- Unpack values from a tuple into separate variables.
- Compare tuple behavior with list behavior.

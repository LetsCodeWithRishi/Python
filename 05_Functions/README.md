# 05_Functions

Functions are reusable blocks of code that make programs easier to organize.

## What you will learn
- Defining functions
- Parameters and arguments
- Return values
- Lambda functions
- Variable scope

## Files in this chapter
- functions.py: function definition and call
- parameters.py: functions that accept values
- return.py: using return values
- lambda.py: short anonymous functions
- scope.py: variable visibility

## Practice ideas
- Create a calculator with functions.
- Write a prime checker.
- Build an Armstrong number checker.
- Create a function to reverse a string.

# 12_File_Handling

File handling allows programs to read and write data from files.

## What you will learn
- Reading files
- Writing files
- Appending to files
- Working with CSV files

## Files in this chapter
- read_file.py: reading text from a file
- write_file.py: writing to a file
- append_file.py: adding content to an existing file
- csv_file.py: creating a CSV file

## Practice ideas
- Create a notes app.
- Save student records to a file.
- Read data from a CSV file.

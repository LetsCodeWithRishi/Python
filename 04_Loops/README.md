# 04_Loops

Loops help repeat code without writing it again and again.

## What you will learn
- for loops
- while loops
- break and continue
- nested loops
- pattern printing

## Files in this chapter
- for_loop.py: simple repeated execution
- while_loop.py: condition-based looping
- break_continue.py: controlling loop flow
- nested_loops.py: loops inside loops
- patterns.py: printing simple star patterns

## Practice ideas
- Print multiplication tables.
- Generate Fibonacci numbers.
- Print a triangle pattern.
- Find prime numbers up to a limit.

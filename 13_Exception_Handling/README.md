# 13_Exception_Handling

Exception handling helps programs respond safely to unexpected errors.

## What you will learn
- try and except blocks
- finally blocks
- custom exceptions

## Files in this chapter
- try_except.py: catching errors
- finally.py: running code regardless of errors
- custom_exception.py: creating a custom exception

## Practice ideas
- Build a safe calculator.
- Handle invalid user input gracefully.
- Read a file safely even when it does not exist.

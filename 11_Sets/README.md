# 11_Sets

Sets are collections of unique values and are helpful when duplicates should be removed.

## What you will learn
- Set creation
- Set operations
- Adding and removing values
- Common set methods

## Files in this chapter
- basics.py: creating a set
- operations.py: union and other set operations
- methods.py: set methods such as add()

## Practice ideas
- Remove duplicates from a list.
- Find common elements between two sets.
- Count unique words in a sentence.

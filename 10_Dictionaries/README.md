# 10_Dictionaries

Dictionaries store data as key-value pairs, which makes them useful for structured information.

## What you will learn
- Creating dictionaries
- Accessing values
- Updating values
- Iterating through dictionaries
- Nested dictionaries

## Files in this chapter
- dictionary.py: basic dictionary example
- methods.py: using dictionary methods
- nested.py: dictionaries inside dictionaries
- iteration.py: looping over dictionary items

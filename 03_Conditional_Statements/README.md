# 03_Conditional_Statements

Conditional statements allow a program to make decisions.

## What you will learn
- if statements
- else statements
- elif statements
- nested conditions

## Files in this chapter
- if.py: basic conditional example
- if_else.py: decision-making with else
- elif.py: multiple conditions
- nested_if.py: conditions inside conditions

## Practice ideas
- Write a voting eligibility program.
- Create a grade calculator.
- Check whether a year is a leap year.
- Find the largest of three numbers.

# Python Learning Repository

Welcome to this Python learning repository. It is organized as a structured roadmap from beginner concepts to small projects.

## How to use this repository
1. Start from the first chapter and move forward in order.
2. Read the chapter README before opening the example files.
3. Run the example scripts with Python to observe the concepts in action.
4. Try the challenge-style exercises and build your own variations.

## Study path
- Basics: variables, input, data types, comments, and type conversion
- Operators: arithmetic, comparison, logical, bitwise, membership, and identity
- Conditionals: if, else, elif, and nested conditions
- Loops: for, while, break, continue, and patterns
- Functions: parameters, return values, lambda expressions, and scope
- Recursion: solving problems by dividing them into smaller parts
- Strings, lists, tuples, dictionaries, and sets
- File handling, exceptions, modules, and object-oriented programming
- Projects: calculators, games, utilities, and management systems

## Suggested workflow
- Read the concepts in each README.
- Run the example files.
- Modify the code and test small variations.
- Complete the exercises to strengthen your understanding.

## Example run
```bash
python 01_Basics/hello_world.py
```

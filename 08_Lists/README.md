# 08_Lists

Lists store multiple values in one variable and are very flexible for day-to-day programming.

## What you will learn
- Creating lists
- Accessing items
- Slicing lists
- List methods
- Nested lists

## Files in this chapter
- list_basics.py: creating and accessing lists
- methods.py: appending and modifying lists
- slicing.py: selecting parts of a list
- nested_lists.py: lists inside lists

## Practice ideas
- Store student marks in a list.
- Build a shopping cart.
- Find the highest mark from a list.
- Sort and reverse a list.

# 01_Basics: Python Basics

This chapter introduces the first building blocks of Python programming.

## What you will learn
- How Python works
- How to run a Python file
- The print() function
- Variables and constants
- Data types such as string, int, float, and bool
- Taking user input
- Writing comments
- Converting data types

## Files in this chapter
- hello_world.py: your first Python program
- variables.py: examples of storing values
- data_types.py: data type demonstration
- input_output.py: reading user input
- type_conversion.py: converting values between types
- comments.py: adding comments to code
- example.py: a simple example script

## Practice ideas
- Create a program that prints your name and age.
- Write a program to calculate the area of a rectangle.
- Convert temperature from Celsius to Fahrenheit.
- Build a small age calculator.
